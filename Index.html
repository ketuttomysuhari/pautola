Budaya & Tradisi

Ruang untuk mendokumentasikan tradisi lokal, ritus, cerita kampung, tokoh adat, serta kalender budaya.

Tradisi Ka Todo

Ka Todo dikenal sebagai tradisi syukur panen/perjamuan bersama yang menguatkan solidaritas sosial. Silakan lengkapi deskripsi sesuai narasi adat dan versi yang disepakati tokoh masyarakat.

Referensi umum tersedia di literatur dan artikel populer (silakan tautkan pada halaman “Dokumen” bila perlu).

Ritual Sepa Api

Sepa Api sering disebut sebagai bagian ritual/atraksi budaya yang merepresentasikan keberanian, ketangguhan, dan simbol penolakan marabahaya. Anda dapat menambahkan dokumentasi foto/video resmi desa.

Contoh publikasi populer: RRI membahas Sepa Api di Pautola. (Opsional untuk dicantumkan di halaman dokumen/berita.)
